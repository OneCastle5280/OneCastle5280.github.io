跑通vue项目所需要配置环境和跑通vue项目

跑通vue项目所需要配置环境

一、安装node.js

1. 到官网进行下载node.js ，推荐下载.msi（一直点next即可）

地址：https://nodejs.org/en/download/

2. 安装成功后，按win + r 输入cmd进入控制台

3. 输入 node -v 检查是否安装成功


显示版本信息即安装成功

二、安装cnpm

1. 安装node.js后，同时也安装了npm。

输入npm -v查看版本号

2. 配置cnpm

但是因为npm有些资源被屏蔽或者是由于国外资源的原因，经常会导致用npm安装依赖包失败的情况。所以我们可以选择使用npm的国内镜像。

等待安装成功后输入cnpm -v查看版本

三、安装webpack

1. 通过npm来安装webpack、

2. 输入webpack -v 来查看是否安装成功

四、安装vue-cli

1. 通过cnpm来安装vue-cli

等待安装成功，没有报错一般就是安装成功
